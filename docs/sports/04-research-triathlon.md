# Triathlon 70.3 / 140.6 Research (canonical data for SportConfig authoring)

## Program length (weeks)
| Distance | Beginner | Intermediate | Advanced |
|---|---|---|---|
| 70.3 | 20–24 | 16 | 12 |
| 140.6 | 24–30 | ~24 | 12 (experienced only) |
Full ≈ double the beginner runway of a half. Optimal 140.6 runway ~1 yr.

## Mesocycle structure
| Phase | Short course | Long course (70.3/140.6) |
|---|---|---|
| Base | ~6 wk | 12+ wk |
| Build | longer OK | shorter for inexperienced long course |
| Peak | 1–2 wk | 1–2 wk |
| Taper | 1–2 wk | 2–3 wk |
3:1 load:recovery mesocycles (2:1 masters/high-fatigue). Full-IM taper math: 75%→50%→25% peak volume over final 3 wk.

## Weekly volume (hours/wk, base→peak) + workouts
| Distance | Level | Hours | Workouts/wk |
|---|---|---|---|
| 70.3 | Beginner | 5–10 | 6–8 |
| 70.3 | Intermediate | 6–12 | 7–9 |
| 70.3 | Advanced | 8–14 | 9–11 |
| 140.6 | Beginner | 6–14 | 6–8 |
| 140.6 | Intermediate | 8–14 | 7–9 |
| 140.6 | Advanced | 8–17 | 9–11 |
Peak: 70.3 ~10–16 h; 140.6 ~15–20 h (traditional) vs 10–14 h (time-crunched/quality-first) — EXPOSE AS A TOGGLE (biggest source disagreement). Volume shape: Base ~60–70% of peak → Build ~80–90% → Peak 100% → Taper −20–70%, stepped on 3:1. Full-IM keeps session COUNT similar to 70.3; extra volume = longer sessions + more bricks/doubles. Sessions: ~2–3 swim, 3 bike (1 long), 3–4 run (1 long, 1 quality).

Practical: 140.6 = S15–25/B45–55/R25–35 (Friel 20/50/30 midpoint); 70.3 = S20–30/B45–50/R25–30. Base most balanced; Build = bike share peaks (runs cut to 15–30 min bricks); Peak = run share up, swim trimmed first. Override: add to weakest discipline (±10–15 pts).

## Experience tiering (per discipline, independent)
**Swim (CSS /100m):** beginner >2:00 or can't swim race dist continuous; intermediate 1:35–2:00 + continuous; advanced <1:35, races swim, OW-comfortable. Anchors: AG median ~2:00 M/~2:10 W; IM top-10% ~1:32–1:35; pro <1:15–1:20. Can't swim distance continuous → beginner regardless of pace.
**Bike (FTP W/kg):** beginner <2.9 M/<2.4 F, can't hold aero >15–20 min; intermediate 2.9–3.6 M/2.4–3.0 F; advanced >3.6 M/>3.0 F (3.5–4.2 = Kona territory). Tri race intensity ~70–76% FTP. Flat course: absolute watts + CdA matter more than W/kg; hills favor W/kg.
**Run (threshold pace + off-bike):** beginner >5:30/km (~8:50/mi) or run/walk, can't run dist off bike; intermediate 4:30–5:30/km, runs dist off bike but slows; advanced <4:30/km, runs strong off bike. Weight "can you run race dist OFF THE BIKE?" heavily. Triathletes run less mileage than pure runners at equal performance.

## Key sessions by discipline/phase
**Swim:** technique/drills (high in base), CSS intervals (10–20×100 @CSS; primary in build), threshold/endurance (2–4k continuous), open-water (high in peak). ~3 swims/wk both distances. CSS zones (offsets to CSS/100m): Easy +8–12s, Steady +6–8s, Moderate +4s, Threshold=CSS, VO2 −4s.
**Bike (%FTP):** endurance/long ride (56–75%), sweet-spot (83–97%, primary FTP builder), threshold/FTP 2–3×15–20 (~95–105%), VO2 2min (115–120%, more 70.3-relevant). 140.6 late build shifts AWAY from sweet-spot → Tempo(Z3)+Z2 durability (race power ~70–76% FTP). Zones %FTP (Coggan): Z1<55, Z2 56–75, Z3 76–90 (SS 88–94), Z4 91–105, Z5 106–120, Z6>120.
**Run:** easy Z2 (~80%), long run, tempo/threshold 20–40 min @race pace, intervals/VO2 (70.3), brick/transition. Keep run most conservative (impact/injury), ramp ≤~10%/wk.

## Bricks (bike→run)
| Phase | Brick run | Off bike | Freq |
|---|---|---|---|
| Early base | 10–20 min easy Z2 | Z2 | 1×/wk |
| Build | 30–45 min tempo | race-effort | 1→2×/wk by wk10–11 |
| Peak | 60–90 min @ IM marathon pace | 4–5 h ride | 2×/wk final 6–8 wk |
70.3: 2h bike→1h run building to 3h→90min. 140.6: 3h→1h @race pace building to 4–5h→90min. Ceilings: 70.3 brick run 40–90 min; 140.6 brick run 60–90 min off 4–5h ride (90 min near-universal ceiling). "4–5h ride → 60–90 min race-pace run" = single most predictive 140.6 session.

## Long-session caps
| | 70.3 | 140.6 |
|---|---|---|
| Long ride | 3–4 h (~50–60 mi) | 5–6 h cap (~80–100+ mi) |
| Long run | 1.5–2 h (~10–13 mi) | 2:30–3:00, ≤20 mi |
| Peak wk vol | 10–12 h | 15–20 h |
IM: never run full marathon in training; several 90 min–2.5 h non-stop runs + 1–2 with walk breaks >3 h; longest run 3–6 wk out. Ride 100+ mi ≥3× before IM. % of race: 140.6 longest ride ~80–90% of race bike, longest run ~55–70% of marathon; 70.3 ride ~100%+, run ~75–100%. Recovery week: 3:1 (2:1 masters), −30% (20–40%), keep some intensity. 140.6 = 2–3 rest days/wk.

## Taper (research: Bosquet 2007, Mujika&Padilla)
Length optimal 8–14 d (~14 for half & full). Volume −40–60% progressive; cut RUN volume least (20–40%), bigger cuts swim/bike. Exponential decay > linear > step. Intensity MAINTAINED. Frequency ≤−20%. 70.3: 2 wk, −30–50% (60–70% race week). 140.6: 2–3 wk (coaches favor 3), ladder 90→70→50→30% peak.

## Load currency (unified TSS)
`TSS = 100 × duration_hours × IF²`, IF discipline-specific: bike(power) NP/FTP; run rTSS = threshold_speed/actual_speed; swim sTSS = swim_speed/CSS_speed; hrTSS fallback (HR vs threshold HR — NOT TRIMP); RPE fallback (RPE×duration ×calibration). Combined weekly load = simple SUM (all same scale). TRIMP stored secondary only, never summed. ACWR acute7:chronic28 (EWMA), sweet spot 0.8–1.3, risk >1.5, on COMBINED load (primary) + per-discipline (localize). Cold-start (~4 wk no history): seed chronic from prescribed load + ramp-rate caps + RPE/readiness. Foster monotony = mean daily load ÷ SD; strain = weekly load × monotony.

## Benchmarks (field tests)
Swim CSS from 400 m + 200 m TT: CSS_speed=(400−200)/(t400−t200) m/s → sec/100m. Guards: t400>t200, plausible (reject <45s or >180s/100m). Bike FTP: 20-min×0.95, OR ramp (~75% 1-min peak/MAP — different formula), OR self-report. No power → bike LTHR + HR zones. Run: reuse existing threshold/VDOT. Per-discipline HR anchors (bike HRmax ~few bpm below run; swim HR unreliable). Provenance: field_test trusted; estimated widens tolerance + retest prompt.

## Fueling / duty-of-care (flags for >5–6 h sessions + 140.6)
Carb g/hr: <75min ~0; 1–2h 30–60; 2–3h 60–90 (multiple-transportable carbs >60); >2.5–3h well-trained ~90 (glc:fru 2:1); gut-trained up to 120. 70.3 (~4–6h): 60–90 g/hr bike. 140.6 (9–17h): 70–90 g/hr, bike = primary fueling window, marathon lower/conservative. Fluid ~500 mL/hr (2–3h) → 500–1000 (3–4h+); sodium 200–1500 mg/hr (huge inter-individual spread → individualize via sweat test). Risks: hyponatremia/EAH (overdrinking; salt tabs don't prevent if overdrink; severe = emergency), GI distress (30–50%), bonking (~90 min glycogen at race intensity). "Train the gut" ≥1 session/wk; never debut race nutrition on race day. Heat: acclimatize 10–14 d, reduce carb/pace. Flags for >5–6h: rehearse race fuel; hydration guardrails both ways (≤2–4% loss); individualize; heat; don't attempt first 5–6h+ unsupported; medical clearance + stop symptoms; GI fallback.

## Intensity distribution
3-zone research model (Z1<LT1, Z2 between thresholds=grey, Z3>LT2) — NOT the 5-zone TP model. ~80/20 easy/hard robust. Polarized (Seiler): ~80/5/15, avoid middle. Pyramidal: ~75/20/5. By phase (long course): Base more polarized → Build/Peak pyramidal (add Z2 race-specificity). By discipline (Plews IM race data): cycling ~25/74/1, running ~4/96/0 (IM run almost all threshold-adjacent → pyramidal late). Swim tolerates most threshold/pyramidal; bike carries most polarized Z3 in base; run kept most easy. Festa 2019 half-IM: Z2 time correlated w/ better run/total in recreational. Hold ~80/20; vary internal Z2/Z3 by phase.

Sources: MyProCoach, Joe Friel, TrainingPeaks, Triathlete (Watson/Hansen), Scientific Triathlon (taper), Jeukendrup/PF&H (fueling), Seiler/EndureIQ/Plews/Festa (intensity), Bosquet 2007, Mujika&Padilla 2003.
