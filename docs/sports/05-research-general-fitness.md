# General Fitness Research (canonical data for SportConfig / general_fitness ProgramType)

## Core problem
Base→Build→Peak→Taper exists to peak on a date. General fitness has no date → Peak/Taper lose their anchor. Replace with continuous cyclical accumulation; progress signal = periodic benchmark re-testing, not a competition. Empirical basis: for non-elite populations no periodization model is clearly superior given progressive overload + variation → pick what's operationally best for open-ended training.

## 1. Periodization model — RECOMMENDED: rotating-emphasis mesocycle blocks (undulating/concurrent inside each)
- Linear/block/undulating are "equally effective" for non-elites; differentiator = competition frequency. General-fitness user "competes every day" (wants capability always) → model that maintains all qualities year-round fits (like an athlete competing 30+×/yr using undulating).
- Replace linear macro-arc with repeating loop of ~3–5 wk emphasis blocks: (1) strength/hypertrophy (cardio at min-effective dose), (2) aerobic/GPP (strength maintenance), (3) mixed power+capacity → repeat, each loop from higher baseline.
- Keep existing rebound/increase/deload microcycles INSIDE each block. Keep 20/60/10/5/5 zone distribution as base state. Demote Peak/Taper to opt-in (if user later adds event). Deload = connective tissue between blocks.
- Sequential emphasis + maintenance of non-emphasized quality beats maxing all simultaneously (avoids interference, faster per-quality progress).

## 2. Concurrent-training interference (modern evidence — reassuring)
- Recent meta (43 studies, 1090 subj): max strength SMD −0.06 (ns), hypertrophy −0.01 (ns) — NOT different from RT alone. Only explosive/POWER reliably blunted: SMD −0.28 (p=0.007).
- Modality/frequency/timing/age did NOT significantly modify strength (recent); older Wilson 2012 found dose-dependent interference (more w/ higher endurance freq/duration, running>cycling). Safe stance: interference real but small, scales with endurance volume/intensity + fatigue overlap.
- Mechanism: endurance→AMPK inhibits mTOR (~3h); strength→mTOR. Basis for spacing.
- ENGINE RULES: (1) prioritized quality first in a same-day session (default strength-first unless block emphasis = endurance); (2) separate hard lift & hard interval by ≥3h ideally a day; (3) same-session cardio after lifting = Z1–Z2 only; (4) keep intervals off heavy-leg days, prefer cycling/rowing cardio on heavy lower days. Encode as scheduling/intensity-pairing, NOT dose cuts.

## 3. Health/longevity weekly targets
- Aerobic: WHO/US 150–300 min/wk moderate OR 75–150 vigorous; "more yields more." Target upper half 250–300 min moderate-equiv for fitness-motivated.
- Strength: ≥2 d/wk all major groups (7-pattern template satisfies this if ≥2×/wk).
- VO2max = highest-value longevity KPI (Mandsager JAMA 2018: elite vs low fitness HR 0.20, no upper limit/plateau; low fitness worse than smoking/diabetes/CAD). Track + re-test.
- Zone-2 vs 4×4 = complementary. Base builds engine (80/20 polarized, easy days truly easy). Norwegian 4×4 (4×4 min @90–95% HRmax, 3 min recovery @60–70%, 2–3×/wk, ~38 min) → +7–10% VO2max/8 wk. Keep base-heavy + ensure ~1 real VO2max session/wk.
- Default weekly dose: aerobic 200–300 min (Z2 ~150–210 = 70–80%, 1 VO2max session 20–40 min hard, optional 1 threshold), strength 2–3 full-body-equiv days ~10 hard sets/muscle/wk, zone dist 20/60/10/5/5.

## 4. Experience tiering (cardio & lifting INDEPENDENT, by training age/rate-of-adaptation)
- Lifting: untrained = <~1 yr consistent progressive loading. Novice PRs session-to-session, intermediate week-to-month, advanced month-to-month (same physiology, different PR cadence). ~1yr orientation: Squat 1.0–1.5×BW, Bench 0.7–1.1×BW, Deadlift 1.25–1.75×BW.
- Cardio: beginner can't run 30 min continuous (run/walk, ~10%/wk progression, capped intensity); intermediate 30+ min / 5–10k comfortable, 1–2 quality/wk; advanced established base, polarized 2–3 quality.
- Program differs by tier: beginner fixed prescriptions + linear + mostly Z1–Z2 + rare deload; intermediate weekly undulation + 1 VO2 + RPE/RIR + deload 4–8 wk; advanced full autoregulation + polarized 80/20 + deload 4–6 wk. Min-effective dose: single-set (7–10 exercises 2–3×/wk) = 2–38% strength gains; 1 session/wk ≥4 exercises improves strength → powers "maintain non-emphasized quality" logic.

## 5. Sub-goals (ASK ONE optional; default balanced) — bias vector, not different engine
| Sub-goal | Bias |
|---|---|
| Body recomp / fat loss | Strength volume HIGH (preserve/build muscle = key lever); cardio moderate for expenditure; flag protein 1.6–2.2 g/kg + modest deficit. Strength 3–4 d/wk, cardio moderate. |
| General strength | Emphasis + volume → lifting; cardio at min-effective dose (Z2 1–2×/wk). Strength 3–4 d/wk, cardio floor. |
| General endurance/GPP | → aerobic volume + 80/20; strength maintenance (2 full-body/wk, even improves economy). Cardio 4–5 d/wk. |
| Balanced (default) | Even split; §3 doses; rotating emphasis, no permanent skew. |
Hard floors on de-emphasized quality (guideline mins). Bias by volume allocation (lower-risk than switching model type).

## 6. Progress mechanism replaces the race
- Continuous cyclical overload + scheduled deloads (every 4–8 wk, 5–7 d, volume −25–90% by fatigue / +RIR).
- RPE/RIR autoregulation (intermediate/advanced) so hard days hard, easy days easy (avoid grey-zone sameness). Rotate exercise variants + cardio modalities across blocks (anti-staleness), patterns constant.
- Benchmark re-test every 8–12 wk at deload boundaries: VO2max estimate / time-trial (1.5-mi/5k/12-min Cooper/submax HR) + est 1RM main patterns + optional body-comp/AMRAP. THE RE-TEST IS THE RACE — PRs/trend lines = motivation + closed loop; drives next block emphasis (VO2 stalled → bias aerobic).

## 7. Example balanced weeks (S=strength, Z2=easy aerobic, VO2=4×4, Thr=threshold)
- 3 d: Mon Full-body S +15–20 Z2; Wed VO2 4×4 + mobility; Fri Full-body S +20 Z2. (≥2 strength + ~150 min aerobic; beginner-friendly)
- 4 d: Mon Upper S +20 Z2; Tue Z2 40–50; Thu Lower S; Sat VO2 or Thr + easy Z2. (2 strength + 2 cardio-led; ~200 min)
- 5 d: Mon Full-body S(push/hinge)+15 Z2; Tue Z2 50–60; Wed Full-body S(pull/squat); Fri VO2 4×4; Sat Z2 45–60. (3 strength-touch, 80/20, ~250–300 min)
- 6 d: Mon Lower S(heavy)+easy spin; Tue Z2 60; Wed Upper S +20 Z2; Thu VO2/intervals; Fri Full-body S(power); Sat long Z2 60–90; Sun rest. (~80% Z1–Z2, 3 strength, heavy legs separated from intervals)
Invariants: strength ≥2×/wk all patterns; aerobic base-heavy (20/60/10/5/5); ≥1 weekly VO2max; avoid hard/hard collisions.

## Summary for engine
Kill countdown → Base permanent, suppress Peak/Taper (opt-in). Rotating ~3–5 wk emphasis mesocycles (strength→aerobic→mixed→repeat) from rising baseline; rebound/increase/deload inside. Interference = scheduling rules not dose cuts (only power at risk). Default upper guideline doses (200–300 min aerobic base-heavy 80/20 + 1 VO2max; strength 2–3 d all patterns ~10 sets/muscle). Tier cardio+lifting independently by training age. Ask one optional sub-goal → volume-allocation bias vector w/ floors. Re-test VO2 + main-lift 1RM every 8–12 wk at deload boundaries = the race.
